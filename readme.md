# Learn Git

------------------------
克隆：`git clone [url]`

----------------
add: `git add *`
commit: `git commit -m 'note'`
push: `git push`

----------------
查看文件状态：`git status`

-------
查看本地分支：`git branch`
查看远程分支：`git branch -r`
创建本地分支：`git branch [name]` ----注意新分支创建后不会自动切换为当前分支
切换分支：`git checkout [name]`
